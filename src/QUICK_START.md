## ğŸ¯ Common Questions

**Q: Is this connected to my real Discord account?**
A: No, by default it uses demo mode with a simulated Discord account.

**Q: Can I use my real Discord?**
A: Yes! See `LOGIN_GUIDE.md` for setup instructions.

**Q: Where is my data stored?**
A: All data is stored locally in your browser (localStorage).

**Q: What happens if I clear my browser data?**
A: All accounts, posts, and settings will be reset.

**Q: Why can I only edit display name once?**
A: This is a permanent restriction to prevent abuse (realistic feature).

**Q: When can I edit my profile picture again?**
A: Every 30 calendar days. A countdown is shown under your avatar.

**Q: What are the warnings for?**
A: They demonstrate the moderation system. Use Admin panel to test.

## ğŸ”§ Troubleshooting

### Login Not Working?
1. Make sure you clicked the Discord button
2. Wait for the 1.5-second loading animation
3. Check for toast notifications
4. Look for error messages in browser console

### Profile Not Loading?
1. Make sure you're logged in (check header buttons)
2. Click "My Profile" (not "Upload")
3. Refresh the page
4. Check browser console for errors

### Can't Upload?
1. Click "Upload" button (not "My Profile")
2. Fill in all required fields
3. Select a file
4. Check file size is reasonable

### Features Locked?
1. This is the 24-hour security lock
2. New accounts have restricted access
3. Wait 24 hours or check lock status in settings
4. Locked features show blinking "LOCKED/24HRs"
